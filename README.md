# MichaelHale.xyz

A personal portfolio website that showcases featured projects, technical skills, and more.

## Stack

- Next.js App Router
- React 19
- Tailwind CSS
- MDX content from `content/work`
- YAML technology metadata from `content/technologies`
- Dockerized production runtime

## Setup

```bash
git clone git@github.com:mjhale/michaelhale.xyz.git
cd michaelhale.xyz
corepack enable
pnpm install
pnpm dev
```

App runs at [http://localhost:3000](http://localhost:3000).

## Scripts

- `pnpm dev`: Sync assets, start local Next dev server with Turbopack
- `pnpm build`: Sync assets, build production app with Turbopack
- `pnpm start`: Sync assets, run production server
- `pnpm lint`: Run ESLint

## Content model

- Work entries: `content/work/**/**/*.mdx`
- Technology records: `content/technologies/*.yml`
- Public assets are generated by `scripts/sync-assets.mjs` into `/public` before dev/build/start

## Environment

Copy `.env.example` values into your deployment environment:

- `SITE_URL`

## Deployment

The application is containerized using Podman and supports multi-platform builds:

```bash
# Build for multiple platforms with dynamic git commit tag
podman build --no-cache \
  --platform linux/amd64,linux/arm64 \
  --manifest ghcr.io/mjhale/michaelhale.xyz:latest .

# Push latest
podman manifest push --all \
  ghcr.io/mjhale/michaelhale.xyz:latest \
  docker://ghcr.io/mjhale/michaelhale.xyz:latest

# Push git commit tag
podman manifest push --all \
  ghcr.io/mjhale/michaelhale.xyz:latest \
  docker://ghcr.io/mjhale/michaelhale.xyz:$(git rev-parse HEAD)

# Run container
podman run -d -p 3000:3000 --env-file .env \
  --name michaelhale.xyz ghcr.io/mjhale/michaelhale.xyz:latest
```
