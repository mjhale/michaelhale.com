# MichaelHale.xyz

A personal portfolio website that showcases featured projects, technical skills, and more.

## Stack

- Next.js App Router
- React 19
- Tailwind CSS
- MDX content from `content/work`
- YAML technology metadata from `content/technologies`
- Dockerized production runtime

## Setup

```bash
git clone git@github.com:mjhale/michaelhale.xyz.git
cd michaelhale.xyz
npm install
npm run dev
```

App runs at [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev`: Sync assets, start local Next dev server with Turbopack
- `npm run build`: Sync assets, build production app with Turbopack
- `npm run start`: Sync assets, run production server
- `npm run lint`: Run ESLint

## Content model

- Work entries: `content/work/**/**/*.mdx`
- Technology records: `content/technologies/*.yml`
- Public assets are generated by `scripts/sync-assets.mjs` into `/public` before dev/build/start

## Contact API

`POST /api/contact/`

Request JSON:

```json
{
  "name": "string",
  "email": "string",
  "message": "string",
  "company": "string (optional honeypot)"
}
```

Response contract:

- `200 { "ok": true }`
- `400 { "ok": false, "errors": {...}, "message": "Invalid payload." }`
- `500 { "ok": false, "message": "..." }`

## Environment

Copy `.env.example` values into your deployment environment:

- `SITE_URL`
- `CONTACT_TO_EMAIL`
- `CONTACT_FROM_EMAIL`
- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_SECURE`
- `SMTP_USER`
- `SMTP_PASS`

## Deployment

The application is containerized using Docker and supports multi-platform builds:

```bash
# Build for multiple platforms with dynamic git commit tag
docker buildx build --no-cache --platform linux/amd64,linux/arm64 \
  -t ghcr.io/mjhale/michaelhale.xyz:latest \
  -t ghcr.io/mjhale/michaelhale.xyz:$(git rev-parse HEAD) .

# Push both tags
docker push ghcr.io/mjhale/michaelhale.xyz:latest
docker push ghcr.io/mjhale/michaelhale.xyz:$(git rev-parse HEAD)

# Run container
docker run -d -p 3000:3000 --env-file .env \
  --name michaelhale.xyz ghcr.io/mjhale/michaelhale.xyz:latest
```
